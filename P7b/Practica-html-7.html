Terminos con:

A B C D E F G H I J K L M N Ñ O P Q R S T U V W X Y Z


Terminos con A

area

Con este elemento podemos considerar una imagen como una serie de porciones. Por ejemplo en el caso de un mapa podemos definir diferentes partes de la imagen como áreas del mismo para establecer ubicaciones de lugares mediante coordenadas.


adress

Muestra la información de contacto de una persona, como por ejemplo, nombre y dirección de correo electrónico, es decir, con este tag, podemos especificar información de contacto relevante sobre la persona, empresa que realiza una web, pero su utilización no está pensada para una dirección física de un lugar, sino más bien para la datos de contacto online.

Terminos con B

body

Este elemento define el cuerpo principal del documento HTML que estamos editando. En HTML 4 este tag tenía la posibilidad de especificar atributos, pero estaban en desuso, mientras que en HTML 5 han sido definitivamente eliminados, por lo que el tag ya no tiene ningún atributo posible.


bdi

El tag bdi permite tratar una parte del texto de forma independiente al resto y darle una dirección diferente. Un uso común de este tag, se da en idiomas como el chino, japonés o el árabe, que emplean un sistema de escritura de texto diferente al occidental.

Terminos con C

canvas

El elemento canvas, nos permite crear un contenedor en el que podemos dibujar cualquier tipo de gráfico, para lo cual se deberá utilizar algún lenguaje de script, como por ejemplo Javascript para definir como se dibujan los gráficos y su comportamiento.


code

El tag code, forma parte de un conjunto de elementos que nos permiten establecer el resaltado de texto como por ejemplo, el de un editor de programación en un documento HTML. En concreto, todo el texto que vaya dentro de una etiqueta code, será considerado como parte del código de un programa.

Terminos con D

dfn

Pertenece al conjunto de elementos que permiten realizar resaltado de texto. Todo el contenido que vaya en esta etiqueta, se considerará la definición de una palabra. Esto es útil a la hora de crear un diccionario o un glosario de terminos.


div

Es el elemnto más utilizado en HTML 4 y sirve para agrupar contenidos. Esto nos permite dar estio de forma sencilla a una sección o grupo de elementos. El tag div suele utilizarse para formar la estructura básica (posición de los diferentes menús, imaágenes, etc). En HTML 5 disponemos del tag section que nos permite dar un significado a las diferentes partes de la página. El principal cambio se encuentra en el atributo align que en HTML 5 ha sido eliminado, quedando en manos del css la alineación de estos elementos.

Terminos con E

em

Dentro de los elementos para el resaltado de texto, es aquél que nos permite marcar un texto con una mayor importancia, pero desde el punto de vista semántico y no desde visual.


embed

Dentro de este elemento podemos insertar contenido externo a la web, como flash, java, etc.

Terminos con F

fieldset

Es el tag que nos permite agrupar diferentes campos de un formulario que estén relacionados entre sí, como por ejemplo, los datos personales en un formulario de registro de usuario. En HTML 5 se han definido nuevos atributos para poder tratar los campos del formulario a nivel de grupo. Estos son: disabled, form y name, que permiten respectivamente, desabilitar un grupo de campos, hacer referencia a uno o varios formularios a los que pertenecen estos campos, y dar un nombre al grupo de campos del formulario.


figcaption

Es un tag complementario a figure que nos permite asignar un título a la imagen que va dentro de este elemento. Podemos situarlo sobre la imagen o bajo ella.

Terminos con G

No existen en html5.

Terminos con H

header

El tag header nos permite definir contenidos como preámbulo, que es muy útil en casos como una noticia, un documento de texto o un artítulo periodístico, así como una entrada de blog. Nos permite insertar texto que describa brevemente el contenido sobre el que va a tratar dicho documento, noticia o entrada. En definitiva, nos permite que un contenido determinado se pueda diferenciar entre la cabecera, que se definiría con este tag y el cuerpo, que por ejemplo se podría definir con un tag article.


h1, h2, h3, h4, h5, h6

Estos elementos nos permiten crear títulos de diferentes tamaños donde h1 es el tamaño mayor, y h6 es el mas pequeño, pudiendo definir títulos generales, títulos de sección, etc. El atributo align ha sido eliminado, ya que estaba en desuso.

Terminos con I

i

Esta etiqueta HTML ha sido utilizada comúnmente para hacer que un texto concreto pase a la llamada letra cursiva o itálica, pero ahora, se emplean las hojas de estilos para formatear las fuentes y el tag i, ha quedado relegado a un uso en caso de que no exita otro que represente mejor lo que queremos definir o para cuando se quiere destacar un tecnicismo o un nombre.


img

Permite la inserción de imágenes de múltiples formatos en la web, como jpg, gif, png, etc. Este tag a penas tiene cambios con respecto a HTML 4. Únicamente, han sido eliminados aquellos atributos que se habían quedado obsoletos en favor de las hojas de estilos como los referentes a al alineación o los bordes. Atributos importantes: src, width, height, alt, que nos permiten indicar la ruta y las dimensiones de la imagne. Ademas, el atributo alt, nos permite fijar un título descriptivo que nos aparecerá con un tooltip al situarnos sobre ella con el ratón.

Terminos con J

No existen en html5

Terminos con K

kbd

Forma parte del conjunto de etiquetas que permiten definir el resaltado de texto. Este tag nos permite tratar un texto como si fuese una entrada de teclado de un programa.


keygen

Este tag se utiliza para el cifrado a la hora de enviar un formulario, creando un generador de claves para el mismo. Este campo nos permite utilizar algoritmos de cifrado como el rsa, dsa o ec para el cifrado del formulario, empleando un sistema de envío de clave pública. Atributos importantes: keytype. Define el algoritmo que se utilizará para el cifrado del formulario. name. Nombre del elemento keygen. chalenge. Indica si la clave debe cambiarse cuando se envia el formulario.

Terminos con L

li

El tag li define un elemento dentro de una lista ya sea ordenada (ol) o desordenada ul. En HTML 5 los atributos type y value han sido eliminados. Por ejemplo, en un índice, los tags li son los elementos que albergan el contenido del índice. A continuación se muestra un ejemplo.


legend

Este elemento nos permite agregar un título que nos puede servir para ofrecer una información adicional sobre un grupo de campos de un formulario fieldset de esta manera podemos por ejemplo agrupar los campos referentes a los datos personales de un usuario que se registra en nuestra web (nombre, apellidos, dirección, etc) con un título que indique: “Datos Personales”. De esta forma queda claramente diferenciado del resto de campos que contenga el formulario de registro. Por otra parte en HTML 5, se puede aplicar el tag legend a otros elementos como figure o details, aunque en el caso del elemento figure, ya disponemos de figcaption para este fin.

Terminos con M

map

Es un tag que permite dividir una imagen en secciones que llevarán a un enlace que aportará mas información. Por ejemplo en un mapa del mundo, podemos establecer secciones a través de coordenadas que definan cada uno de los continentes, donde cada continente tiene un enlace que nos lleve a un mapa con todos los países que alberga ese continente. Atributos importantes: name.


mark

Es el elemento que nos permite marcar una parte del texto con un color de subrayado emulando a los rotuladores de marcado que se utilizan sobre papel.

Terminos con N

nav

Define una parte de la web como un menú de navegación, es decir, dentro de este tag, se agrupa una serie de enlaces que generalmetne suele ser el menú de secciones de la web, también conocido como menú de navegación.


nonscript

Este elemento tiene su principal utilidad para ofrecer una alternativa para aquellos usuarios que tengan un navegador que no soporte lenguajes como JavaScript. Esto permite mostrar contenido en dichos casos y también evita que en caso de que no se soporten los lenguajes de script, se muestre el código del script en la pantalla lo que supone un riesgo de seguridad importante.

Terminos con Ñ

No existen en html5

Terminos con O

option

Este es el tag que se utiliza en los combos select para definir las diferentes opciones del listado. En HTML 5, también se usa en el tag datalist. Atributos importantes: label, selected, value.


output

A través de este tag, podemos mostrar el resultado de una operación que realice un script, como puede ser un calculo de cambio de moneda, una ecuación matemática o una conversión de unidades de medida. Atributos importantes: for. Establece la relación existente entre los operandos que realizan el cálculo y el resultado devuelto por éstos. name. Indica el nombre del tag output.

Terminos con P

p

Este es un tag de tratamiento de texto que nos permite crear parrafos en HTML. Además, por defecto, los tags p, provocan un salto de linea forzado al final y al principio del párrafo. En HTML 5, la alineación de párrafo se está permitida, pero lo más recomendable es el uso de css para este fin.


pre

Este elemento nos permite insertar texto formateado en HTML, es decir, todo el texto que se introduzca dentro de este tag, se mostrará tal y como lo hemos escrito en el documento HTML, respetando los saltos de linea, espacios y demás características del texto.

Terminos con Q

No existen en html5

Terminos con R

rt

Este tag está pensado para ser utilizado junto al tag ruby como complemento descriptivo de la pronuciación de los caracteres del continente asiático


ruby

Se emplea junto con los tags rt y rp para escribir caracteres orientales. Por ejemplo, es útil para una definición de un carácter asiático dentro de un diccionario con su pronunciación.

Terminos con S

sumary

Este elemento se combina con el tag details para crear secciones desplegables dentro de la web. Por ejemplo se puede utilizar para crear un desplegable que contiene el resumen de una noticia o las características de un producto. Se debe tener en cuenta que este tag sólo lo soporta Google Chrome actualmente, por lo que en el resto tendremos que utilizar JavaScript para emular este efecto.


s

Este tag hace referencia a texto incorrecto. Por ejemplo un uso que se le puede dar a este elemento es el de la corrección de exámenes online, para que el alumno sepa que partes del mismo no son correctas y que partes sí lo son. El efecto visual es el mismo en HTML 5 que en HTML 4, pero mientras que en HTML 4 se le daba un uso para tachar partes del texto, en el caso de HTML 5, tiene su utilidad para el texto que no es correcto dentro de un contexto concreto, es más una redefinición semántica que de funcionalidad, ya que el efecto es el mismo.

Terminos con T

th

Similar al tag td, pero para filas que se definen en la cabecera de una tabla. Este tag tambien ha sufrido cambios en cuanto a sus atributos y tan sólo conserva los siguientes: colspan, rowspace y scope.


thead

Es el elemento que define el contenedor para la todo lo que tenga relación con la cabecera de la tabla. Los atributos referentes a este elemento han sido eliminados.

Terminos con U

u

Al igual que el tag s, este tag ha adquirido un significado en HTML 5, de tal forma, que ahora se emplea para subrayar texto que está mal formado o escrito de forma incoherente. En HTML 4 se empleaba para subrayar texto (simulaba en la web la opción de subrayado de un editor de textos). Ahora con el enfoque semántico, se le ha dado un uso similar al tag s.


ul

Este tag representa una lista que no tiene un criterio de orden de esquema. A diferencia de las listas generadas con el tag ol, no siguen ningún criterio de orden preestablecido y su forma de representar los elementos de la lista es más apropiada para casos como por ejemplo, las características de un producto, las ventajas e inconvenientes de una decisión empresarial, etc. Los atributos compact y type que HTML 4 ya estaban en desuso, por el hecho de que se daba estilo a la lista mediante css, han sido definitivamente eliminados. Estos atributos se empleaban para reducir el tamaño de una lista y para dar una forma concreta a la hora de su visualización respectivamente (circular, cuadrada o en forma de disco).

Terminos con V

var

Este tag representa el contenido como una variable de un lenguaje de programación. Pertencece al conjunto de elementos que se emplean para el resaltado de texto.


video

Hasta el momento los tags object e iframe eran la única herramienta de la que se disponía para insertar video en la web, lo cual nos obligaba a depender, o bien de un servidor de vídeo externo, como Youtube o Vimeo, o bien emplear flash para insertar un reproductor en la web que nos permitiera reproducir el vídeo en nuestro propio servidor.

Terminos con W

wbr

Es un tag cuya principal utilidad es indicar en que parte de un texto sería recomendable añadir un salto de linea, para una mejor lectura del mismo, o para darle una sintaxis más correcta.
